ກ່ຽວກັບລະບົບ LaoStraw IoT

ແອັບພິເຄເຊີນ ຫຼື ລະບົບ StrawIoT ແມ່ນສ້າງຂື້ນມາເພື່ອໃຊ້ງານສະເພາະດ້ານໃນວຽກງານ IoT ຂອງໂຮງເຮືອນສະຕຣໍເບີຣີອັດສະລິຍະ ພາຍໃຕ້ໂຄງການ IoTCloudServe@TIEN ແລະ ເປັນບົດໂຄງການຈົບຊັ້ນປະຣີນຍາຕິຂອງນັກສືກສາ ມະຫາວິທະຍາໄລແຫ່ງຊາດ ຄະນະວິສະວະກຳສາດ ສາຂາວິຊາ ວິສະວະກຳຄອມພິວເຕີ

ທິມພັດທະນາລະບົບ

1. ທ. ປິຕິ ພັນທະສົມບັດ ( Development WebApp & MobileApp )
2. ທ. ສິນເກີດ ພຸດທະວົງ ( Development Arduino C Programming )
3. ທ. ປາຫວິດ ຈັນທະວົງ ( Development Design 3D Material )
